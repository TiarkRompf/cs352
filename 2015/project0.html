Project Preliminaries

If you use the CS lab machines to work on your project, or remote into the CS department, the following setup is recommended so that you can easily get to the javacc installation in /homes/cs352. Add the following line to the end of your .bashrc file:

    source /homes/cs352/SETUP.sh

This will add the JAVACC and CS352 environment variables and modify your PATH so that that the "javacc" command will work from your shell. Alternatively, you may download and install javacc yourself from the JavaCC website. Your submission will be graded using JavaCC 6.0, which of the time of this writing was the latest version.
